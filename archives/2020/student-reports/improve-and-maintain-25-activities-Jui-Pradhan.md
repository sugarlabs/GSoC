--------------------------
***Google Summer of Code, 2020***

*Name - Jui Pradhan*

*Email - juipradhan2k@gmail.com*

*GitHub username - JuiP*

**Project:**

My work includes feature additions, bug fixes, adding support for different display resolutions, few design changes, port to Python 3 as a part of improving Sugar Activities. This project also aimed at maintaining activities, the work for which included: making activity releases, adding or fetching translations, flake8 changes and testing activities extensively. Moreover, fixing issues, reviewing pull requests and finishing unmerged work was also a part of maintenance.

**Technologies used:**

The major part of the project involved coding in Python 3 and the use of TelepathyGLib, GLib, GObject, Flake8, Git, PyGame, Gtk+3 and python3-coverage package. Porting also involved the use of 2to3 Python port tool.
